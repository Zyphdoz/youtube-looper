### Youtube video looper
A simple tool to loop specific sections of a YouTube video.

I made this tool many years ago to practice [chorusing](https://www.youtube.com/watch?v=m5JwiNSIHxY) but I never shared it publicly until now because it was hastily put together so the user experience was not as polished as I would have liked it to be. Nowadays, I've come to realize that something, even if it isn't perfect, is still better than nothing at all. So I decided to put it up here and host it on github pages in case it ends up being useful to someone. https://zyphdoz.github.io/youtube-looper/
